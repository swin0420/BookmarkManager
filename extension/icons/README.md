# Extension Icons

Add PNG icons in these sizes:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

You can use any bookmark icon. The extension will work without custom icons but will show a default puzzle piece icon.

## Quick Fix

You can generate simple icons using any image editor or online tool like:
- https://favicon.io/
- https://realfavicongenerator.net/

Or use a simple 16x16, 48x48, and 128x128 blue square as a placeholder.
